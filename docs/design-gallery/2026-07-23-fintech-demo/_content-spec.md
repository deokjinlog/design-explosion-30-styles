## 데이터 규칙
- 브랜드·가맹점·사용자 전부 가상(핀그리드·새벽마켓·데일리박스·페이앤고·미니샵 코리아·김운영). 실존 금지. 숫자·문구 한 글자도 안 바꿈.
- **금액은 전부 `₩` 접두로 통일. `원` 접미 금지.** (예: `₩1,284,732,050,000` — `…원`으로 바꾸지 말 것)
- 차트·스파크라인·도넛·진행바 전부 **라이브러리 금지 → 순수 CSS·SVG**. JS 실패해도 최종 상태가 HTML에 남아야 함.
- **대시보드형 승부처: 숫자 타이포 · 카드 경계 · 데이터 밀도 · 차트 색 운용.** (원형 A: `archetypes.md` A절)
