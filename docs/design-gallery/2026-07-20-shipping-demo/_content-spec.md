# [콘텐츠 스펙 — 배송 추적] · 세션 고정 · 원형 E 폼·입력형 (송장 조회 + 배송 타임라인)

> `Service > Shipping` (domains.md **E→A→F**). 조회 폼(E)으로 들어가 타임라인(F 성격)으로 끝나는 **특이 조합**.
> rubric: 즉시성·명료 1순위. ⛔ KPI 카드·도넛·목표 진행률 금지(대시보드 아님). E 필수 region: form.

## 화면 구성 — data-region 부착
### 1. 상단 정체성 `data-region="identity"`  로고 **어디쯤** + `내 배송함` + 아바타
### 2. 조회 폼 `data-region="form"` (주인공)
- 운송장 번호 `4821-9930-1174` (입력됨, 완료 상태)
- 택배사 `자동 감지 ▾` (드롭다운)
- **에러 시연**: 두 번째 입력칸 `1234` — `운송장 번호는 12자리예요`
- **비활성 시연**: `수취인 이름` 필드 (택배사 자동감지 시 비활성)
- 주요 버튼 `조회` + 보조 `여러 건 조회`
### 3. 배송 요약 `data-region="summary"`
`한빛택배 · 4821-9930-1174` · 상품 `무선 이어폰 외 1건` · 도착 예정 **오늘 18시~20시** · 받는 곳 `서울 강남구 (문 앞)`
### 4. 배송 타임라인 (5단계, 현재=4번째) — 각 단계: 시각 · 위치 · 설명 · 상태
| 시각 | 위치 | 설명 | 상태 |
|---|---|---|---|
| 07-18 14:20 | 이천 허브 | 집화 완료 | 완료 |
| 07-19 02:05 | 옥천 HUB | 간선 상차 | 완료 |
| 07-19 09:41 | 강남 물류센터 | **분류 지연 — 폭우로 인한 간선 도착 지연이 있었으며 당일 배송 물량이 이월되어 순차 처리 중입니다** ← 아주 긴 설명 | 지연 |
| 07-20 08:12 | 강남2 배송점 | 배송 출발 (기사 김○○) | 진행중 |
| — ← 미래(결측) | 수취인 | 배송 완료 | 대기 |
상태 4종: `완료` / `진행중` / `지연` / `대기`
### 5. 최근 조회 목록 (F 성격, 3건)
`4821-9930-1174 · 오늘 도착 예정` / `7719-0021-8845 · 배송 완료(어제)` / `2210-5567-0093 · 조회 결과 없음`
### 6. 액션 (공통 `action`)  `알림 받기` 토글(켜짐) + 보조 `주소 변경 요청`
### 7. 빈 상태 `data-region="empty"`  `운송장 번호를 찾을 수 없어요` → `번호를 다시 확인하거나 택배사를 직접 선택해보세요.` → `택배사 선택`

## 견고성/규칙
에러 필드 · 비활성 필드 · 아주 긴 설명(타임라인) · 미래 단계 결측 `—` · 지연 상태 · 조회 실패.
