# [콘텐츠 스펙 — 연서(yeonseo) v2] · 세션 고정 · 절대 변경 금지

> **화면 원형: B 대화형(주) + C 읽기형(보조)** — `references/archetypes.md`
> v1 은 대시보드 원형을 잘못 이식해 KPI·도넛·목표달성률이 들어갔다. **이번엔 그 넷이 없다.**
> 이 파일은 모든 시안이 **글자 하나 다르지 않게** 공유한다. 스타일만 변수다.

## 결정 기준 (rubric)

- **브랜드 성격**: 조용하고 사려 깊음. 문학의 품위는 있되 학술적으로 무겁지 않음.
- **타겟**: 사랑이 어려운데 자기가 뭐가 어려운지 모르는 20~30대.
- **1순위 목적**: **신뢰** (조언을 받아들일 만한 상대로 보이는가) > 차별화 > 전환.
- 제품 원칙: 답을 주지 않고 다른 사람의 이야기를 준다 / 훈계하지 않는다 / 무게중심은 내적.

## ⛔ 이 원형에 넣으면 안 되는 것 (archetype-lint FAIL 대상)

**KPI 지표 카드 · 도넛/구성비 차트 · 목표 진행률/달성률 · 스파크라인 · 추이 라인차트.**
연서는 사람을 측정하는 제품이 아니다. 소양을 KPI로 만들면 제품 철학과 정면으로 어긋난다.

## 화면 구성 (모든 스타일 공통)

### 1. 상단 정체성 바
좌: 제품명 **연서** · 우: 보조 버튼 `모아보기` + 아바타 이니셜 `ㅇ`
(장식 최소 — 대화가 주인공이므로 상단바가 시선을 뺏으면 안 된다)

### 2. 좌측 — 지난 대화 (내비게이션)
`오늘` 아래 2건 · `지난주` 아래 2건, 각 항목은 그날의 첫 마디 일부 + 날짜
- `연락이 오면 반갑고 안 오면…` · `자꾸 먼저 연락하게 돼요` · `헤어질 때가 된 걸까요` · `좋은 사람인데 설레지 않아요`
현재 대화가 선택 강조. 900px↓ 접힘.

### 3. **대화 흐름 — 화면의 주인공** (전체 폭, 위에서부터 쌓임, 화자 구분)

1. **연서**: `요즘 사랑에서 어떤 게 제일 어려우세요?`
2. **나**: `연락이 오면 반갑고 안 오면 하루가 다 망가져요. 제가 너무 매달리는 것 같기도 하고, 사실은 그냥 확인받고 싶은 건지도 모르겠어요.` ← **아주 긴 발화**(줄바꿈 견고성)
3. **연서**: `기다림이 힘든 건가요, 기다리는 자신이 싫은 건가요?`
   → 보기 버튼 2개: `[ 기다림이 힘들다 ]` `[ 그런 내가 싫다 ]`
4. **나**: `후자요.` ← **아주 짧은 발화**(극단 대비)
5. **연서** (작성 중): 기다리는 상태 표시 — 점 3개든 무엇이든 스타일 언어로

### 4. 진단 (연서가 짚어준 결핍 — 200자 이내, NFR-4)
> `확인받아야 안심되는 마음이 보여요. 없는 건 상대의 애정이 아니라 스스로에 대한 확신 같습니다.`
축 배지 `내적` 동반.

### 5. **스토리 카드 (읽기형 요소 — 두 번째 주인공)**
- 바탕 고전: **오만과 편견** · 제인 오스틴
- 축 배지: `내적`
- 본문 (**긴 텍스트가 주인공**, 읽기 타이포가 승부처):
  `엘리자베스는 그를 오만하다고 단정했다. 첫인상은 선명했고, 선명한 만큼 의심할 이유가 없었다. 그러나 그 확신은 그녀가 그를 알아갈수록 조금씩 어긋났다. 편지 한 장이 도착했을 때, 그녀가 흔들린 것은 그의 해명이 아니라 자신이 그토록 확신했다는 사실이었다. 사람을 잘못 본 것보다 무서운 건, 잘못 볼 수 있다는 걸 몰랐다는 쪽이다. 그녀는 그날 이후 자신의 판단을 한 번 더 들여다보는 사람이 되었다.`
- 교훈 한 줄: `첫인상은 자주 틀린다 — 나에 대한 첫인상도 마찬가지다.`
- 하단: `다른 이야기도 들어볼래요?`

### 6. 입력창 (하단 고정, NFR-8)
placeholder `편한 만큼만 답해도 됩니다` + 주요 버튼 `보내기`

### 7. 안내 배너 (info)
`결핍을 짚으려면 두세 마디가 더 필요해요.`

### 8. 되돌아보기 — 받은 스토리 (밀도 지점 + 상태 4종)
축 필터 칩: `전체`(선택) · `외적` · `내적`
목록 6건 — 작품 · 작가 · 축 · 상태 (상태는 **색 + 텍스트 병행**):

| 작품 | 작가 | 축 | 상태 |
|---|---|---|---|
| 오만과 편견 | 제인 오스틴 | 내적 | 들음 |
| 위대한 개츠비 | F. 스콧 피츠제럴드 | 외적 | 보류 |
| **어느 시골 사제의 일기에서 사랑에 대해 남긴 긴 각주와 그 해설** ← 아주 긴 제목 | — ← 결측 | 내적 | 다시듣기 |
| 폭풍의 언덕 | 에밀리 브론테 | 내적 | 들음 |
| 시라노 드 베르주라크 | 에드몽 로스탕 | 외적 | 새로움 |
| 안나 카레니나 | 레프 톨스토이 | 내적 | 보류 |

### 9. 빈 상태 (축 필터 결과 0건)
헤드라인 `아직 이 축의 이야기가 없어요` → 설명 `대화를 두어 마디 더 나누면 맞는 이야기를 찾아드릴게요.` → CTA `대화 이어가기`

## 데이터 규칙
- 작품·작가는 **저작권 만료 고전(실존)**. 스토리 본문은 창작(NFR-6).
- 사용자는 이니셜·역할 라벨만. 숫자·문구는 **한 글자도 바꾸지 않는다.**
- **읽기형 승부처는 타이포그래피다** — 본문 측정폭·행간·인용 처리·제목 위계로 스타일이 갈린다.
