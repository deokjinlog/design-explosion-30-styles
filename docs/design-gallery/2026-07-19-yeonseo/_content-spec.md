# [콘텐츠 스펙 — 연서(yeonseo)] · 세션 고정 · 절대 변경 금지

> 이 파일은 30개 시안이 **글자 하나 다르지 않게 공유**하는 콘텐츠다. 스타일만 변수다.
> 대시보드형이 어색한 도메인이라 **"연서 워크스페이스"** 화면으로 등가 치환했다(컴포넌트 밀도는 유지).

## 결정 기준 (rubric)

- **브랜드 성격**: 조용하고 사려 깊음. 문학의 품위는 있되 학술적으로 무겁지 않음.
- **타겟**: 사랑이 어려운데 자기가 뭐가 어려운지 모르는 20~30대.
- **1순위 목적**: **신뢰** (조언을 받아들일 만한 상대로 보이는가) > 차별화 > 전환.
- 제품 원칙: 답을 주지 않고 다른 사람의 이야기를 준다 / 훈계하지 않는다 / 무게중심은 내적.

### 11. 대화 패널 (활동 피드 대체 — 이 제품의 정체성)
위에서부터 쌓인 대화 4마디, 화자 구분:

1. **연서**: `요즘 사랑에서 어떤 게 제일 어려우세요?`
2. **나**: `연락이 오면 반갑고 안 오면 하루가 다 망가져요. 제가 너무 매달리는 것 같기도 하고, 사실은 그냥 확인받고 싶은 건지도 모르겠어요.` ← 아주 긴 발화(줄바꿈 견고성)
3. **연서**: `기다림이 힘든 건가요, 기다리는 자신이 싫은 건가요?` + 보기 버튼 2개 `[ 기다림이 힘들다 ]` `[ 그런 내가 싫다 ]`
4. **나**: `후자요.` ← 아주 짧은 발화(극단 대비)

이어서 **진단 + 스토리 제안 카드 1개**:
- 진단: `확인받아야 안심되는 마음이 보여요. 없는 건 상대의 애정이 아니라 스스로에 대한 확신 같습니다.`
- 스토리 카드: 축 배지 `내적` / 바탕 고전 `오만과 편견 · 제인 오스틴` / 교훈 한 줄 `첫인상은 자주 틀린다 — 나에 대한 첫인상도 마찬가지다.`
- 하단: `다른 이야기도 들어볼래요?`

### 12. 빈 상태 (한 영역에서 시연 — 모아보기 축 필터 결과 0건)
