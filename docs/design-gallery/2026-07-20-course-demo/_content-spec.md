로고 **배움터** + 브레드크럼: `프론트엔드 마스터` › `3. 상태관리` + 우측 `12 / 48강 완료`(카운트, 진행률% 아님)
### 2. 비디오 캔버스 `data-region="canvas"` (주인공, 가장 큰 자리)
현재 강의 재생 중: 제목 오버레이 **3-4. useReducer로 복잡한 상태 다루기 — 언제 useState를 넘어서야 하는가**  (← 아주 긴 제목)
