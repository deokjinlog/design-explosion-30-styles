# [콘텐츠 스펙 — 지식베이스 검색] · 세션 고정 · 원형 H 검색·결과형

> 사내 지식베이스/도움말 검색. **검색어 하이라이트가 주인공**. rubric: 명료·속도 1순위. 스타일만 변수.
> H 필수 region: search · results. 권장: facets · empty. ⛔ KPI·도넛·목표 진행률 금지.

## 화면 구성 — data-region 부착
### 1. 검색바 `data-region="search"` (주인공, 쿼리 입력됨)
로고 **헬프센터** + 검색창 `환불 정책` (입력됨) + `검색` · 결과 요약 `"환불 정책" 검색결과 34건 (0.12초)`
### 2. 좌측 패싯 필터 `data-region="facets"` (3그룹)
- 카테고리: 결제(12) · 계정(8) · 배송(7) · 기타(7)
- 문서 유형: 도움말 · FAQ · 공지
- 최신순 · 관련도순(선택)
### 3. 결과 목록 `data-region="results"` (6건, 각 제목·발췌·하이라이트·출처·수정일)
1. **환불 정책** 전문 안내 — `상품 수령 후 7일 이내 <mark>환불</mark> 신청이 가능하며, 단순 변심의 경우 <mark>환불</mark> 배송비가…` · 도움말 · 3일 전
2. 부분 <mark>환불</mark>은 어떻게 처리되나요? — `일부 품목만 <mark>환불</mark>할 때는 남은 주문 금액 기준으로…` · FAQ · 1주 전
3. **정기결제 해지 후 남은 기간에 대한 일할 계산 환불과 자동 재청구 중지 처리 절차 상세 안내** ← 긴 제목 — `해지 시점부터 다음 결제일까지…` · 도움말 · 2주 전
4. <mark>환불</mark> 처리 기간 안내 — `카드사 영업일 기준 3~5일…` · FAQ · — ← 수정일 결측
5. 해외 결제 <mark>환불</mark> 시 환율 — `<mark>환불</mark> 금액은 <mark>환불</mark> 시점 환율로 재계산되어…` ← 하이라이트 다중 · 공지 · 1개월 전
6. 포인트로 결제한 주문의 <mark>환불</mark> — `포인트는 즉시 복원되고 현금 결제분만…` · FAQ · 2개월 전
### 4. 관련 검색어 (공통 `action`)  `환불 배송비` · `환불 기간` · `부분 환불` + 페이지네이션 `‹ 1 2 3 … 6 ›`
### 5. 빈 상태 + 오타 교정 `data-region="empty"`
`'환불정책'에 대한 결과가 없어요` → `혹시 이걸 찾으셨나요? "환불 정책"` → CTA `문의 남기기`

## 견고성/규칙  아주 긴 제목·하이라이트 여러 개(5번)·출처 결측 `—`·결과 0 빈 상태·오타 교정. 실존 서비스 금지. 숫자·문구 한 글자도 안 바꿈. 승부처: **검색어 하이라이트 처리·발췌 타이포·결과 구분**.
