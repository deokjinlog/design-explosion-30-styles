# [콘텐츠 스펙 — 계정 설정] · 세션 고정 · 원형 K 프로필·설정형

> 계정/환경설정 화면(Enrico Profile+Settings). rubric: 명료·신뢰 1순위. 스타일만 변수.
> K 필수 region: settings-list. 권장: profile · nav · action. ⛔ KPI·도넛·차트·목표 진행률 금지.
> 승부처: **행 밀도·구분선·토글/컨트롤 형태·그룹 구분·위험 구역**.

## 화면 구성 — data-region 부착
### 1. 좌측 설정 내비 `data-region="nav"`  `프로필(선택) · 알림 · 보안 · 결제 · 연동 · 고급`
### 2. 프로필 헤더 `data-region="profile"`  아바타 이니셜 `ㅈ` + 이름 `정하람` + 역할 `일반 회원` + `사진 변경`
### 3. 설정 목록 `data-region="settings-list"` (그룹 3~4, 각 행: 레이블+설명+컨트롤)
**계정**
- 이름 `정하람` (값 → 편집 링크)
- 이메일 `haram@example.com` · `인증됨` 배지
- 언어 `한국어 ▾` (드롭다운)
**알림**
- 이메일 알림 `켜짐` (토글 ON)
- 마케팅 수신 `꺼짐` (토글 OFF)
- 방해 금지 시간 `설정 안 됨` — ← 값 결측
**보안**
- 2단계 인증 `사용 안 함` (토글 OFF) + 설명 `로그인 시 추가 인증을 요구해요`
- 비밀번호 `마지막 변경 3개월 전` → 변경 링크
- 로그인 기기 `기기 3대` → 관리 링크 (비활성: 확인 중)
### 4. 위험 구역 (그룹 하단, 강조)
`계정 삭제` — `삭제하면 모든 데이터가 사라지고 되돌릴 수 없어요.` + `계정 삭제` 버튼(위험색)
### 5. 저장 상태 (공통 `action`)  `모든 변경사항 저장됨 · 방금 전` + 주요 `변경사항 저장`

## 상태 4종 (컨트롤/행 상태)
`켜짐`(토글 ON) / `꺼짐`(토글 OFF) / `비활성`(확인 중 행) / `위험`(삭제) — 색+텍스트 병행.

## 견고성/규칙
값 결측(방해 금지 `설정 안 됨`) · 비활성 행 · 위험 액션 강조 · 아주 긴 설명 문구.
이름·이메일 가상(example.com). 숫자·문구 한 글자도 안 바꿈.
